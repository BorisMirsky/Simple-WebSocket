Библиотеки:
- websockets - для небольших проектов, 
- aiohttp — относитильно тяжёлая и функциональная, позволяет совмещать HTTP и websocket 


Плюсы веб сокетов
- Сокращение нагрузки на сервер.
- Мгновенная двусторонняя связь (дуплекс) клиент-сервер в реальном времени.
- Подходит для веб-, мобильных и десктопных приложений. 
- Безопасность соединения обеспечит протокол wss:// вместо ws://.
- Высокая пропускная способность такой интеграции. 


Потенциальные минусы
- При большом количестве одновременных постоянных подключений 
может возникнуть гонка за ресурсы, обращающиеся к одному сокету сервера. 
- Если нужна устойчивая система, то придется управлять 
состоянием и синхронизацией между клиентом и сервером в рамках
длительного постоянного соединения.

Как пользоваться
1. Переходим в Simple-WebSocket\myvenv\Scripts, в CLI выполняем:
   activate

2. В качестве редактора кода для простых случаев использую IDLE.
   Вызов из папки src:
   python -m idlelib.idle

3. Запуск
   Нужны 2 окна терминала (cmd, psh). В обоих запускаем вирт. окружение и переходим в папку src.
   - python server.py
   - python client.py
